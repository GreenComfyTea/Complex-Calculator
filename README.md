# Prerequisites
- .Net Framework 4.7.2

# Features
- Unary operations (applies to the first operand):
  - Conjugation;
  - Recipropal;
  - NOP (doesn't produce a result, but calculates additional information about the first oeprand);
- Binary operations:
  - Addition;
  - Subtraction;
  - Multiplication;
  - Division;
  - Exponentiation.
